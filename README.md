# Mon Portfolio

Mon espace pour présenter mes projets et partager ma vision du développement web.

Je suis développeur fullstack orienté produits, avec une forte appétence pour la création d'applications web modernes. Je crée des applications utiles et performantes, du backend à l'interface utilisateur, en privilégiant la performance, la simplicité et une expérience fluide.

Ce portfolio a été construit avec **React**, **Vite** et **Tailwind CSS** — exactement les technologies que j'adore utiliser pour concevoir des interfaces claires et efficaces.

## 🚀 Démarrage

### Prérequis
- Node.js (v14 ou supérieur)
- npm ou yarn

### Installation

1. Cloner le dépôt
```bash
git clone https://github.com/USER/REPO.git
cd REPO
```

2. Installer les dépendances
```bash
npm install
```

3. Lancer le serveur de développement
```bash
npm run dev
```

Le site sera accessible à `http://localhost:5173`

## 🔨 Commandes disponibles

- `npm run dev` — Lance le serveur de développement
- `npm run build` — Crée la version de production
- `npm run preview` — Prévisualise la build production localement
